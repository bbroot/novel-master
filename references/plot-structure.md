# 小说结构参考

## 出版级三幕结构

### 第一幕：建立（占全文15%）

| 节点 | 内容 | 典型字数 |
|------|------|---------|
| 日常世界 | 主角的正常生活 | 全书的5% |
| 激励事件 | 打破日常的意外 | 全文的10% |
| 拒绝改变 | 主角的第一次抗拒 | 全文的12% |
| 接受冒险 | 第一扇门关闭（不可逆选择） | 全文的15% |

### 第二幕：对抗（占全文70%）

| 节点 | 内容 | 典型字数 |
|------|------|---------|
| 新世界 | 进入"另一边" | 全文的20% |
| 试炼与盟友 | 学习规则，结识伙伴 | 全文的25% |
| 中点转折 | 假胜利或假失败 | 全文的50% |
| 反派逼近 | 敌人动用真正实力 | 全文的60% |
| 一切尽失 | 低谷，主角濒临放弃 | 全文的70% |
| 重新出发 | 获得新认知，迎接最终决战 | 全文的75% |

### 第三幕：解决（占全文15%）

| 节点 | 内容 | 典型字数 |
|------|------|---------|
| 最终决战 | 与反派正面交锋 | 全文的82% |
| 高潮 | 核心冲突爆发 | 全文的85% |
| 结局 | 问题解决，新秩序建立 | 全文的92% |
| 尾声 | 象征性收束 | 全文的100% |

## 非线性结构变体

### 套盒叙事（Russian Doll）
- 章节内嵌故事
- 例：《看不见的客人》式多版本叙事
- 技巧：每层叙事揭示真相，同时推翻上一层

### 螺旋式推进
- 同一事件，从多个角色视角重复叙述
- 每次重复增加新信息，视角人物内心世界逐渐暴露
- 适用于：悬疑小说、群像小说

### 冰山叙事
- 读者只看到海面上的10%（正文）
- 背后90%的真相由作者掌握，通过细节暗示
- 回收时产生最大情感冲击

## 章节节奏设计

### 章末钩子类型
1. **信息钩**：揭示一个新事实
2. **情感钩**：在情感最高点戛然而止
3. **行动钩**：正当行动进行到一半时结束
4. **悬疑钩**：问出一个问题
5. **反讽钩**：用一句双关或讽刺收尾

### 每章场景数建议
- 2场景章：紧凑型，用于快节奏推进
- 3场景章：标准型，平衡叙事与节奏
- 4+场景章：复合型，用于人物内心戏或复杂事件

## 分卷核心原则

每卷需要：
1. 一个独立的核心问题（贯穿本卷）
2. 本卷内的小高潮（卷末）
3. 连接前后卷的钩子（卷末留给下一卷）
