# 类型小说专项参考

## 悬疑小说

### 核心公式
**悬疑 = 信息差 + 节奏控制**

读者知道的 = 主角知道的 + 作者刻意透露的
读者不知道的 = 真相 - 主角知道的

### 叙事结构选择
- **古典推理**：读者与侦探同时获得线索，公平竞技
- **硬汉派**：主角不知道，读者半知半解
- **黑色悬疑**：全知叙事，真相比主角知道的更多
- **叙述性诡计**：欺骗读者（视角欺骗/时间欺骗）

### 悬疑章节节奏
```
第1章：命案/危机发生 → 高压开篇
第2-3章：初步调查 → 信息铺陈
第4-N章：线索→误区→新线索→新误区 → 螺旋上升
高潮前章：所有线索汇总 → 读者推理窗口
高潮章：真相揭示 → 反转或确认
```

### 悬疑对话准则
- 警察/侦探说话：精确、简短、追问
- 嫌疑人说话：防御性、回避关键、过多解释（心虚表现）
- 证人说话：带情绪、个人视角、不自觉的谎言

### 线索密度
- 每3000字至少埋1个有效线索
- 核心线索提前出现（在读者注意力之外）
- 红鲱鱼（误导线索）：每章不超过1条

## 言情小说

### 核心公式
**言情 = 阻碍 + 情感积累 + 爆发**

### 关系发展四阶段
1. **邂逅**：不愉快的第一印象（最佳开局）
2. **冲突**：立场/身份/价值观对立
3. **暧昧**：靠近-后退-再靠近的反复
4. **确定**：一个必须面对的危机

### 言情禁忌
- 前三章就确定关系（无阻碍，不好看）
- 无波澜的甜蜜（没有张力）
- 第三者强行介入（机械降神）

### 言情场景类型（必须覆盖）
- 四目相对（电光火石）
- 肢体接触（心跳加速）
- 困境互助（情感升温）
- 深夜对话（内心袒露）
- 误会争吵（关系危机）
- 复合重逢（高潮情感）

### 心理描写尺度
- 言情小说心理描写比悬疑重要
- 但心理描写不能代替行动
- 最佳：心理+行动的同步呈现

## 科幻小说

### 科幻设定三原则
1. **自洽**：设定内部逻辑一致
2. **必要**：所有设定都服务于主题
3. **留白**：不解释过多，留给读者想象

### 科幻叙事
- 信息解释：融入情节（对话/文件/实验）
- 绝不用大段说明文
- 让读者和主角一起发现

### 世界构建清单
- [ ] 物理规则（与现实的差异）
- [ ] 社会结构
- [ ] 技术水平
- [ ] 普通人如何看待新技术
- [ ] 新技术带来的新问题

## 历史小说

### 历史感营造
- 用细节，不用大词
- 服饰/饮食/语言的时代感
- 但人物思维不能完全古代化（否则读者无法共情）

### 历史真实与虚构
- 大事件严格依史实
- 小人物完全虚构
- 虚构人物参与大事件但不改变大事件结果

## 现实主义小说

### 写作核心
- 观察力 > 想象力
- 细节密度决定真实感
- 社会问题不是口号，是人物困境

### 出版级现实主义标准
- 无符号化人物（穷人/富人/官员/工人不是脸谱）
- 矛盾有层次（一件事有多重原因）
- 结局不廉价（不给简单答案）
