# 出版级文风指南

## 叙述语气

### 叙事视角选择
**默认：第三人称限知视角**
- 读者随主角一同发现真相
- 作者不能比主角知道得更多（除非刻意设计）
- 心理描写：仅限主角或明确限知视角人物

**慎用：第一人称**
- 适合：内心独白型、回忆体、私密情感类
- 风险：视角封闭，难以处理多线叙事
- 禁忌：第一人称下不能写其他人的内心（除非超自然设定）

### 叙事节奏

| 场景类型 | 句式节奏 | 字数/分钟阅读量参考 |
|---------|---------|-------------------|
| 动作场面 | 短句为主，急促 | 800字/分钟 |
| 对话场面 | 长短交替，口语 | 500字/分钟 |
| 内心独白 | 长句为主，缓慢 | 300字/分钟 |
| 场景描写 | 短中句，五感 | 400字/分钟 |

**核心原则：** 一章内必须有至少一次节奏切换（慢→快或快→慢）。

## 句式规范

### 禁忌句式（必须避免）
- "突然……"（除非情节必要）
- "莫名……"（同上）
- "绝对……"（过于绝对，缺乏文学性）
- "于是……"（叙事衔接用"接着"、"然后"更自然）
- "此时此刻"（陈词滥调）
- "他不自觉地……"（不自觉是个谎言，人物所有行为都是自觉的）

### 句式变换技巧
```
他走到窗边，看着外面的雨。（陈述）
雨落在窗玻璃上。他在看。（静物+短句，节奏放慢）
这雨已经下了三天。够久了。（评论+短句，制造张力）
```

## 对话写作

### 对话四项基本原则
1. **每句对话必须有推进功能**：要么揭示信息，要么改变关系，要么制造冲突
2. **口语化但有个性**：不同教育背景的人说不同的话
3. **潜台词优先**：对话表面意思≠实际意思
4. **动作描写嵌入对话**：让人物"动起来"，而非静止对话

### 对话格式
```markdown
"你还好吗？"她问。              ← 先说后问（标准）
她问："你还好吗？"                ← 先问后说（强调动作）
"你还好吗？"她问。她不敢看他。   ← 双动作（强调紧张感）
"你还好——"他打断她。            ← 说话中断（制造紧张）
```

### 对话中的沉默
沉默比对话更有力：
```markdown
"……"
"什么？"
"没什么。"
```
省略号是最低级的沉默写法。用动作填满沉默：
```markdown
她等他说下去。他把头转向窗外。
茶凉了。
```

## 描写技巧

### 五感描写原则
- 视觉：最常用，但最廉价
- 听觉：被低估，往往更能调动情绪
- 嗅觉：最强唤起记忆的能力
- 触觉：制造亲密感
- 味觉：被严重低估

### "展示而非告知"示例

| ❌ 告知 | ✅ 展示 |
|--------|--------|
| 他很紧张 | 他把烟灰缸里的烟蒂一根根捻灭 |
| 她很伤心 | 她把菜打包带回家——平时她从不打包 |
| 房间很冷 | 呼出的气在灯下变成一团白雾 |

### 形容词使用规则
- 名词+动词 > 形容词+名词
- 一句话不超过2个形容词
- 避免堆砌颜色词（除非象征意义）

## 章节收尾

### 五种强力章末

1. **断裂式**（最推荐）
正当读者以为故事会往A方向发展时，突然切断，切换场景。

2. **回声式**
结尾呼应章节开头，用同一意象但不同语境，形成回响。

3. **悬停式**
在主角即将做出关键决定时停止，留下悬念。

4. **顿悟式**
主角突然想通了某件事（但不说出答案），戛然而止。

5. **静默式**
在一个情感高点突然安静下来，让情绪自行沉淀。

## 禁忌词表

```
突然、莫名、突然地、猛然、骤然
绝对、永远、完全、所有、一切
于是、接着、然后（过渡性用词）
此时此刻（陈词滥调）
我看见/我听到（无谓的叙述者声音）
不自觉地（人物行为动机模糊）
很、非常、特别、极其（弱化形容）
```

## 文字密度标准

出版级文字密度要求（每千字）：
- 有效信息点：≥8个（场景/人物/事件的关键信息）
- 对话占比：30-50%
- 描写占比：20-40%
- 叙述占比：20-40%

低于此标准 → 水文 / 堆砌
高于此标准 → 信息过载 / 阅读疲劳
