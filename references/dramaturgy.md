# 经典剧作与叙事理论

## 一、三幕结构的经典理论

### 悉德·菲尔德（Syd Field）三幕结构
> 《电影剧本写作基础》—— 影视/小说通用

```
第一幕 0% ───────────── 25% ───────────── 50% ───────────── 75% ───────────── 100%
      │                   │                   │                   │
    建置              建构              中点             高潮
    (Setup)         (Confrontation)      (Midpoint)      (Climax)
      │                   │                   │                   │
    日常→激励→拒绝    障碍→失败→更大障碍    转折/假胜利    最后一搏→解决
```

**关键节点：**
| 节点 | 位置 | 内容 |
|------|------|------|
| 激励事件 | 第12-15% | 打破主角生活平衡 |
| 第一转折点 | 第25-27% | 主角接受挑战，不可回头 |
| 中点 | 第50% | 假胜利或假失败，虚实转换 |
| 第二转折点 | 第75% | 反派动用全力，低谷 |
| 第三转折点 | 第88% | 主角获得新认知 |

### 布莱克·斯奈德（Blake Snyder）《救猫咪》
> 15个节拍表（Beat Sheet）

| 节拍 | 位置 | 说明 |
|------|------|------|
| 1.开场画面 | 0-1% | 定调，定主题，定基调 |
| 2.主题呈现 | 5% | 用一句话暗示主题 |
| 3.设定 | 1-10% | 主角日常世界 |
| 4.催化剂 | 12% | 改变一切的事件 |
| 5.争论/抗拒 | 12-25% | 主角拒绝新使命 |
| 6.第二幕开始 | 25% | 进入新世界 |
| 7.B故事 | 22-27% | 爱情线/支线，往往是真正主题 |
| 8.游戏 | 25-50% | 最有趣的部分，观众被吸引 |
| 9.中点 | 50% | 虚假的胜利或失败（外部）/ 亲密时刻（内部）|
| 10.坏人逼近 | 50-75% | 外在障碍+内在疑虑同时升级 |
| 11.一切尽失 | 75% | 假失败，"死亡与婚礼" |
| 12.黑鸟时刻 | 75-80% | 最低点，主角几乎放弃 |
| 13.第三幕开始 | 80% | 顿悟，获得新方案 |
| 14.结局 | 80-99% | 问题解决，主题闭环 |
| 15.终场画面 | 99-100% | 与开场镜像，验证变化 |

**五大型类型（Five Genre）：**
1. **怪物在门廊**（Monster on the Run）：惊悚/恐怖
2. **落入情网**（Fall in Love）：爱情
3. **哥们有难**（Buddy Love）：友情/公路
4. **愚者成功/愚者失败**（Dummy Luck/Dummy Lost）：傻小子/灰姑娘
5. **被制度化**（Institutionalized）：反乌托邦/组织阴谋

## 二、麦基《故事》核心理论（出版级最高参考）

### 三层故事设计
1. **思想层**（Theme）：你想说什么
2. **故事层**（Story）：发生了什么
3. **场景层**（Scene）：具体如何呈现

### 麦基七大关键概念

**1. 主控思想（Controlling Idea）**
一句话=价值观 + 原因
例："爱能够超越阶级鸿沟，因为人类情感的真实无法被社会规则所封印"

**2. 人物真相（Character Truth）**
人物真相 = 在压力下做出的选择（而非他的想法或背景）
> 压力越大，人物越真实

**3. 人物弧光（Character Arc）**
```
内在真相揭露
     ↑
    / \
   /   \
  /     \
 /       \
/_________\  ← 平衡线（日常世界）
```
最佳弧光：主角在追求外在目标的过程中，被迫改变内在信念

**4. 激励事件（Inciting Incident）**
- 必须发生在主角身上（第10-15%）
- 必须有一个具体的、清晰的"无法撤回的"选择
- 必须把主角赶出平衡状态

**5. 对抗力量（Opposition）**
- 每一次推进必须有等量或更大的反推进
- 对手实力必须等于或超过主角
- "让主角的痛苦真实"

**6. 场景分析（Beat）**
每个场景必须：
- 有有意义的转折（不是无聊的重复）
- 满足"展示代价"：读者感受到每一步的代价

**7. 亚类型（Subgenre）**
| 类型 | 核心问题 |
|------|---------|
| 爱情 | 两个人如何学会彼此理解？ |
| 怪物 | 非我族类，其心必异？ |
| 大团圆 | 好人应有好报？ |
| 救赎 | 恶人能变成善人？ |
| 惩罚 | 作恶者必须付出代价？ |
| 教育 | 人生的意义是什么？ |

## 三、英雄之旅（Joseph Campbell / Christopher Vogler）

12阶段英雄之旅，适用于任何有"成长主线"的小说：

| 阶段 | 名称 | 说明 |
|------|------|------|
| 1 | 普通世界 | 主角的日常 |
| 2 | 冒险召唤 | 命运敲门 |
| 3 | 拒绝召唤 | 恐惧/留恋 |
| 4 | 遇见导师 | 给予知识/道具 |
| 5 | 跨越门槛 | 不可回头 |
| 6 | 试炼/盟友/敌人 | 学习规则 |
| 7 | 接近洞穴 | 面对最大挑战 |
| 8 | 严峻考验 | 死亡体验/内心最大恐惧 |
| 9 | 获得奖赏 | 解决/和解 |
| 10 | 归来之路 | 余波/追兵 |
| 11 | 复活 | 最终考验，脱胎换骨 |
| 12 | 带回灵药 | 将改变带回普通世界 |

**主角vs反英雄：**
- 经典英雄：主动接受命运
- 反英雄：被动卷入，最终被迫成长
- 现代写作倾向于反英雄（更真实）

## 四、悬念大师课（悬疑专项）

### 希区柯克悬念公式
> "炸弹不能爆炸，悬念才成立"

```
陈词滥调版：坏人A→发现炸弹→爆炸→恐慌
         （恐慌10秒）

希区柯克版：观众看到炸弹在桌下
         → 主角们聊日常
         → 观众每一秒都在紧张
         （悬念持续整部电影）
```

### 悬念的三种类型
| 类型 | 定义 | 适用场景 |
|------|------|---------|
| 情节悬念 | 接下来会发生什么 | 动作/危机 |
| 性格悬念 | 角色会如何选择 | 伦理困境/成长 |
| 知识悬念 | 读者比角色知道得多 | 悬疑/惊悚 |

### 阿加莎·克里斯蒂方法
- **叙述性诡计**：欺骗读者对事实的认知（时间/视角/身份）
- **公平线索**：所有破案线索必须提前出现
- **最后逆转**：读者重读时发现所有线索都指向另一方向

### 悬疑章节节奏公式
```
第X章结尾：
  提出一个读者无法回答的问题
  → 读者必须继续读下去找答案

答案揭晓时：
  立刻抛出下一个问题（更大/更紧迫）
  → 永不断裂的悬念链
```
