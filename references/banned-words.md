# 出版级小说质量标准自检

## 章节完成自检

完成每章后，对照以下清单逐一检查：

### 基础质量
- [ ] 无错别字（使用搜索功能检查常见错字）
- [ ] 无病句（主谓宾完整，无成分残缺）
- [ ] 无重复用词（同一段落内相同词汇不出现超过2次）
- [ ] 无逻辑漏洞（时间线/地理线/人物线一致）

### 叙事质量
- [ ] 视角统一（单章内严格单一视角，无跳视角）
- [ ] 无"万能叙述者"（作者代替人物观察/感受）
- [ ] 展示而非告知（所有评价通过细节呈现）
- [ ] 对话推进剧情（无闲聊填充字数）

### 节奏质量
- [ ] 场景数合理（2-4个场景，不超过5个）
- [ ] 有节奏变化（慢场景后接快场景）
- [ ] 章节有钩子（结尾留悬念/情感高点）
- [ ] 无流水账（每个场景必须有存在的理由）

### 伏笔质量
- [ ] 无新增悬而未决的问题（除非刻意设计悬念章）
- [ ] 伏笔出现有合理语境（不突兀）
- [ ] 字数在目标范围内（±10%）

## 常见问题修复

### 问题：水文/堆字数
修复：删除所有非必要场景描写，对话精简30%

### 问题：人物扁平
修复：给人物增加一个与其核心性格矛盾的行为

### 问题：情节推进慢
修复：每3000字必须有一个明确的事件进展

### 问题：章节结尾平淡
修复：最后一段删除，改为对话中断或动作中停顿

## 出版级达标线

达到以下标准，可以认为章节质量合格：

| 维度 | 标准 |
|------|------|
| 字数控制 | 目标±10% |
| 错字率 | < 0.5‰（千分之零点五）|
| 有效信息密度 | ≥ 8信息点/千字 |
| 对话推进比 | ≥ 60%（对话中有推进的句子占比）|
| 展示/告知比 | ≥ 70% |
| 伏笔命中率 | ≥ 95%（回收率）|
